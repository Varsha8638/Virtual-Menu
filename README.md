# ✨ Aggarwal Sweet Center — Virtual Menu

A beautifully crafted **interactive virtual menu experience** designed to bring the warmth of traditional flavors into a modern digital space.

Built with pure frontend technologies, this project transforms a regular restaurant menu into an elegant and immersive browsing experience with smooth animations, premium styling, and responsive design.

---

## 🍽️ About The Project

This virtual menu was created to provide customers with a visually engaging way to explore dishes online.

Inspired by premium restaurant aesthetics, the design focuses on:

✦ Elegant typography  
✦ Smooth user interactions  
✦ Luxury-inspired dark theme  
✦ Mobile-first responsiveness  
✦ Simple and enjoyable browsing experience  

---

## 🌟 Features

🍴 Interactive food categories  
🎨 Premium UI with modern aesthetics  
📱 Fully responsive layout  
✨ Smooth animations and transitions  
👨‍🍳 Custom chef illustrations  
🏷️ Special dish highlights  
⚡ Splash screen experience  
⬆️ Scroll-to-top interaction  
🌙 Immersive dark theme
